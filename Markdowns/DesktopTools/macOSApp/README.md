# macOS版 デスクトップツール

最終更新日：2024/4/1

## 概要
PC環境から、nRF5340基板の動作に必要な各種設定／動作テスト等を行うためのツールです。

### 動作環境
macOS Monterey (Version 12)〜 <br>
最終更新日現在、Intel mac版のみ用意しています。Apple siliconには未対応です。

## 機能

* ペアリング実行
* ペアリング解除要求
* ファームウェア更新
* PINGテスト実行
* バージョン参照
* Flash ROM情報参照
* 現在時刻参照／設定
* ログファイル参照
